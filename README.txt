Gestor Inteligente de Clientes (GIC) - Módulo 4

Cómo ejecutar
1) Abrir una terminal en esta carpeta.
2) Ejecutar: python main.py

Rutas de archivos (quedan dentro del proyecto)
- Importar: datos/clientes_entrada.csv
- Exportar: datos/clientes.csv
- Reporte: reportes/resumen.txt
- Log: logs/app.log

Notas
- El email se usa como identificador único para evitar duplicados.
- Tipos válidos: regular / premium / corporativo
